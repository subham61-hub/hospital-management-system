# Hospital Management System (Updated)

## Quick Start
1. Double-click `open_hms.bat` — starts server + opens browser
2. Or double-click `HMS_Localhost.url` shortcut (server must be running)
3. Register/Login as doctor/patient
4. Book appointments, manage dashboard
